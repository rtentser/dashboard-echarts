# Типы диаграмм и компонентов

## Диаграммы

### 1. `single-value` - Одно значение

Простое отображение одного числового значения с опциональной единицей измерения.

```json
{
  "title": "Текущий остаток",
  "type": "single-value",
  "value": 125000,
  "unit": "₽"
}
```

**Поля:**
- `title` (обязательно): название метрики
- `value` (обязательно): числовое значение
- `unit` (опционально): единица измерения

---

### 2. `summary` - Сводка

Отображает текущее значение с изменением относительно исходного значения.

```json
{
  "title": "Остаток денег",
  "type": "summary",
  "currentValue": 125000,
  "previousValue": 100000,
  "unit": "₽"
}
```

**Поля:**
- `title` (обязательно): название метрики
- `currentValue` (обязательно): текущее значение
- `previousValue` (обязательно): исходное значение для сравнения
- `unit` (опционально): единица измерения

---

### 3. `bar-chart` - Столбчатая диаграмма

Классическая столбчатая диаграмма.

```json
{
  "title": "Продажи по месяцам",
  "type": "bar-chart",
  "dataSource": {
    "xField": "общее.месяц",
    "yField": "ДДС.Остаток денег на конец месяца"
  }
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `dataSource` (обязательно):
  - `xField`: путь к данным для оси X (из mock.json)
  - `yField`: путь к данным для оси Y (из mock.json)

---

### 4. `stacked-bar-chart` - Столбчатая диаграмма с накоплением

Столбчатая диаграмма с несколькими series, сложенными друг на друга.

```json
{
  "title": "Структура расходов",
  "type": "stacked-bar-chart",
  "x": ["Янв", "Фев", "Мар"],
  "series": [
    { "name": "Категория 1", "data": [100, 150, 200] },
    { "name": "Категория 2", "data": [80, 120, 160] }
  ]
}
```

**Поля:**
- `x`: массив категорий для оси X
- `series`: массив series с name и data

---

### 5. `line-chart` - Линейный график

График с линией, соединяющей точки данных.

```json
{
  "title": "Тренд продаж",
  "type": "line-chart",
  "dataSource": {
    "xField": "общее.месяц",
    "yField": "ДДС.Поступления по ОД"
  }
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `dataSource` (обязательно):
  - `xField`: путь к данным для оси X
  - `yField`: путь к данным для оси Y

---

### 6. `area-chart` - Диаграмма с областями

Линейный график с закрашенной областью под линией.

```json
{
  "title": "Накопление капитала",
  "type": "area-chart",
  "dataSource": {
    "xField": "общее.месяц",
    "yField": "ДДС.Остаток денег на конец месяца"
  }
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `dataSource` (обязательно):
  - `xField`: путь к данным для оси X
  - `yField`: путь к данным для оси Y

---

### 7. `pie-chart` - Круговая диаграмма

Классическая круговая диаграмма.

```json
{
  "title": "Распределение доходов",
  "type": "pie-chart",
  "labels": ["Источник A", "Источник B", "Источник C"],
  "values": [30, 45, 25]
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `labels`: массив названий секций
- `values`: массив значений для каждой секции

---

### 8. `doughnut-chart` - Кольцевая диаграмма

Круговая диаграмма с отверстием в центре.

```json
{
  "title": "Доля рынка",
  "type": "doughnut-chart",
  "labels": ["Компания A", "Компания B", "Компания C"],
  "values": [35, 40, 25]
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `labels`: массив названий секций
- `values`: массив значений для каждой секции

---

### 9. `pie-3d-chart` - Объёмная круговая диаграмма

Круговая диаграмма с визуальным 3D эффектом.

```json
{
  "title": "Распределение ресурсов",
  "type": "pie-3d-chart",
  "labels": ["Ресурс 1", "Ресурс 2", "Ресурс 3"],
  "values": [40, 35, 25]
}
```

**Поля:**
- `title` (опционально): заголовок графика
- `labels`: массив названий секций
- `values`: массив значений для каждой секции

---

### 10. `combo-chart` - Комбинированная диаграмма

Комбинация столбчатой и линейной диаграмм на одном графике.

```json
{
  "title": "Продажи и прибыль",
  "type": "combo-chart",
  "x": ["Янв", "Фев", "Мар", "Апр"],
  "bar": [100, 120, 140, 160],
  "line": [50, 65, 85, 110]
}
```

**Поля:**
- `x`: массив категорий для оси X
- `bar`: массив значений для столбцов
- `line`: массив значений для линии

---

### 11. `scatter-chart` - Диаграмма рассеивания

График с отдельными точками, показывающий корреляцию между двумя переменными.

```json
{
  "title": "Корреляция X и Y",
  "type": "scatter-chart",
  "x": [10, 20, 30, 40, 50],
  "y": [5, 15, 25, 35, 45]
}
```

**Поля:**
- `x`: массив значений для оси X
- `y`: массив значений для оси Y

---

## Контейнеры для группирования

### `group-horizontal` - Горизонтальная группировка

Размещает компоненты горизонтально в одной строке.

```json
{
  "type": "group-horizontal",
  "title": "Ключевые метрики",
  "items": [
    { "type": "chart", "config": { "title": "Метрика 1", "type": "single-value", "value": 100 } },
    { "type": "chart", "config": { "title": "Метрика 2", "type": "single-value", "value": 200 } }
  ]
}
```

**Поля:**
- `title` (опционально): заголовок группировки
- `items`: массив компонентов (chart, tabs-block, group-horizontal, group-vertical)

---

### `group-vertical` - Вертикальная группировка

Размещает компоненты вертикально с единым заголовком для всей группы.

```json
{
  "type": "group-vertical",
  "title": "Финансовый отчёт",
  "items": [
    { "type": "chart", "config": { "title": "Доход", "type": "bar-chart", "dataSource": {...} } },
    { "type": "chart", "config": { "title": "Расход", "type": "bar-chart", "dataSource": {...} } }
  ]
}
```

**Поля:**
- `title` (обязательно): заголовок группировки
- `items`: массив компонентов (chart, tabs-block, group-horizontal, group-vertical)

---

## Компоненты верхнего уровня

### `chart` - Компонент диаграммы

Оборачивает одну диаграмму.

```json
{
  "type": "chart",
  "config": {
    "title": "Название диаграммы",
    "type": "bar-chart",
    "dataSource": { "xField": "...", "yField": "..." }
  }
}
```

---

### `tabs-block` - Блок с вкладками

Группирует диаграммы по вкладкам.

```json
{
  "type": "tabs-block",
  "title": "Аналитика",
  "tabs": [
    {
      "name": "Продажи",
      "charts": [
        { "title": "По месяцам", "type": "bar-chart", "dataSource": {...} }
      ]
    },
    {
      "name": "Расходы",
      "charts": [
        { "title": "По категориям", "type": "pie-chart", "labels": [...], "values": [...] }
      ]
    }
  ]
}
```

**Поля:**
- `title` (опционально): заголовок блока вкладок
- `tabs`: массив вкладок с name и charts

---


## Источник данных

Все диаграммы, использующие `dataSource`, загружают данные из:
- **mock.json** (по умолчанию, контролируется переменной `VITE_USE_MOCK=true`)
- **API** (если `VITE_USE_MOCK=false`, используется URL из `VITE_API_URL`)

Поля в `dataSource` используют dot-notation для доступа к вложенным свойствам:
- `"общее.месяц"` → `data.общее.месяц`
- `"ДДС.Остаток денег на конец месяца"` → `data.ДДС["Остаток денег на конец месяца"]`
